RUNTIME POLICY

Free tools win by default.

Default

Zero cost

The registry always prefers FREE and LOCAL tools over API ones, so a run costs nothing unless you say otherwise.

Optional

Bring your own key

Set GROQ_API_KEY in .env for LLM-written scripts. No key, no problem — it falls back automatically.

Always

Permissive only

Every wired-in dependency is MIT, LGPL, or public domain. Tracked in tools/LICENSES.md.

QUICKSTART

From clone to first render.

# 1. install
pip install -r requirements.txt
pip install piper-tts
cd remotion-composer && npm install && cd ..

# 2. add your free Groq key (optional)
cp .env.example .env

# 3. run it
python -m orchestrator.pipeline_engine pipelines/documentary.yaml --topic "city life in the rain"
